# HyperHop — 8BitOps Engine v0.2.4 Notes

## v0.2.4 changes

- Reduced gameplay from 5 lanes to 3 lanes for cleaner mobile readability and tighter arcade feel.
- Moved Pause to the top HUD.
- Removed the bottom Pause button.
- Settings, Sound, and Back to Arcade are now pause-only chrome. They appear while paused or inside Settings, not during live play.
- Settings now supports mobile taps directly on menu rows and swipe controls, so difficulty/background can be changed and Back is reachable without keyboard up/down.
- Kept the repeatable per-difficulty spawn tracks and progressive rank escalation.

## Current identity

HyperHop is the 8BitOps lane-dodger: three lanes, repeatable routes, escalating rank, clean-combo rewards, shield clarity, and Phase Warp as the survival move.


## v0.2.4 — Level Backdrops

- Backdrops are no longer selectable in settings.
- The visual environment now changes by rank: Starfield, Particle Rain, Matrix Rain, Rift Grid, then Neon Storm.
