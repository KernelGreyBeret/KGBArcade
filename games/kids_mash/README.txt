Kid Jam Beat Party PWA

This is the upgraded Kid Jam build with one-tap pop/techno/EDM beat buttons, horn keys, game keys, funny SFX, recording, and offline PWA support.

Deploy the contents of this folder together, for example to /games/kid-jam/ on GitHub Pages.

Required files:
- index.html
- manifest.webmanifest
- service-worker.js
- icons/

PWA install/offline works only when served over HTTPS or localhost. It will not install from a local file:// open.
On Android Chrome: open the deployed page, tap the browser menu, then Install app or Add to Home screen.
