Tommy the Tank — Production README

This package was wrapped by KGB Studio v1.0.11 using the Existing Game Release Wrapper.

What this means:
- The original game files were preserved.
- KGB Studio generated release metadata around the existing game.
- This wrapper does not reverse-engineer the game into KGB Studio actors/scenes.
- PWA conversion was not enabled for this package.

How to test locally:
1. Unzip the package.
2. Open a terminal in the unzipped folder.
3. Run: python3 -m http.server 8000
4. Open: http://localhost:8000/smoke-test.html
5. Confirm PASS.
6. Open: http://localhost:8000/index.html
