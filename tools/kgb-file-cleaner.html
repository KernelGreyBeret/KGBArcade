Strip / Replace
Filesystem reality: a browser cannot rewrite the original Android/macOS/Windows owner, ACL, full path, or creation date fields. This creates new cleaned files inside a new ZIP; the device that saves the ZIP assigns that new ZIP its own filesystem properties.
