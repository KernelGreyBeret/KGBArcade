KGB TAIO Help

This refresh keeps TAIO as a free, local, single-file browser toolkit. Most tools work fully offline after the page loads. HTTP GET depends on browser CORS.

Navigation: Use the left tool list or search bar. On mobile, tap Tools to show/hide the tool list.

Regex Extractor: enter a pattern and source text, then Run. It highlights matches and lists captures. Save Pattern stores it in localStorage.

Line Ops: GREP filters lines; AWK extracts delimited fields; CUT extracts character ranges. Use this for logs, CLI output, and quick data surgery.

JSON Ops: Format, minify, repair common malformed JSON, flatten object paths, filter key/value pairs, and generate a practical schema.

Converters: Convert JSON/CSV/XML/plain lines and HTML/Markdown. The XML converter is intentionally lightweight for common data shapes.

Compare: Line Diff shows added/removed/unchanged lines. Common Data parses JSON/CSV/list-like inputs when possible and finds shared keys/values.

File Viewers: Load or drag files. CSV renders as a table; Markdown renders with a safe lightweight renderer.

Encoding + Hashing: Base64, URL, HTML, and SHA hashing use browser APIs. MD5 was intentionally not carried forward because WebCrypto does not support it and it is legacy/weak.

Scratch: persistent local pad for notes and staging text between tools.
