PrettyForge

Paste or load messy HTML and turn it back into readable, indented source. It can also format embedded CSS and simple embedded JavaScript blocks. Everything runs locally in your browser.

Use case: this is the reverse-side utility for your static-tool workflow. Use Code Shield when you want a protected public build; use PrettyForge when you want readable code again for review, cleanup, learning, or editing.
